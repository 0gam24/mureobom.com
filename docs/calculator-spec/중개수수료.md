# 계산기 명세: 부동산 중개수수료

## 1. 개요
- **슬러그**: `/계산기/중개수수료`
- **카테고리**: 부동산 / 세금
- **타깃 페르소나**: A (부동산 거래 직전자), F (임대차 관계자)
- **1차 키워드**: "중개수수료 계산기"
- **2차 키워드**: 부동산 복비, 중개보수 요율, 9억 이상 중개수수료
- **우선순위**: MVP #8

## 2. 사용자 가치
"매매·전세·월세 계약 직전, 한도 요율과 협의 구간을 한 번에 확인하고 상한액과 협의 가능 범위를 파악"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 거래 유형 | radio | 매매·교환 / 전세 / 월세 |
| 물건 종류 | radio | 주택 / 오피스텔(주거용, 전용 85㎡ 이하) / 그 외(토지·상가·원룸 등) |
| 매매가 / 전세보증금 | number + 단위버튼 | 억 / 천만 / 백만 |
| 보증금 | number | 월세일 때만 |
| 월세 | number | 월세일 때만 |
| 부가세 (VAT 10%) 포함 | checkbox | 기본 해제 |
| 협의 요율(%) | number | 상한 요율 이하에서 사용자가 협의한 실제 요율. 공란이면 상한요율 적용 |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 거래금액 | 매매가 / 전세보증금 / (보증금 + 월세×100 또는 ×70) |
| 적용 상한 요율 | 법정 상한 요율(%) |
| 한도액 | 해당 구간 한도액(소액 구간만) |
| **상한 중개수수료** | 거래금액 × 상한요율 (한도액 존재 시 Math.min) |
| 협의 요율 반영 수수료 | 협의 요율 입력 시 |
| 부가세 | 수수료 × 10% (체크 시) |
| **총 지급액** | 수수료 + 부가세 |
| 양측 합계 (매도+매수) | 참고용 2배 |

## 5. 공식 근거
- 공인중개사법 §32 (중개보수·실비)
- 공인중개사법 시행규칙 §20 (중개보수 요율 및 한도액)
- 국토교통부 고시 「부동산 중개보수 요율」
- 2023.10.19 개정 이후 현행 요율 유지 (2026년 기준 동일)

## 6. 현행 요율표 (주택)

### 매매·교환
| 거래금액 | 상한 요율 | 한도액 |
|---|---|---|
| ~5천만 | 0.6% | 25만 원 |
| 5천만~2억 | 0.5% | 80만 원 |
| 2억~9억 | 0.4% | - |
| 9억~12억 | 0.5% | - |
| 12억~15억 | 0.6% | - |
| 15억 이상 | 0.7% | - |

### 임대차(전세·월세)
| 거래금액 | 상한 요율 | 한도액 |
|---|---|---|
| ~5천만 | 0.5% | 20만 원 |
| 5천만~1억 | 0.4% | 30만 원 |
| 1억~6억 | 0.3% | - |
| 6억~12억 | 0.4% | - |
| 12억~15억 | 0.5% | - |
| 15억 이상 | 0.6% | - |

### 오피스텔(전용 85㎡ 이하, 주거용)
- 매매·교환: 0.5%
- 임대차: 0.4%
- 한도액 없음

### 그 외(상가·토지·비주거용 오피스텔·원룸)
- 0.9% 이내 협의 (상한만 존재, 구간 없음)

### 월세 거래금액 산식
- 기본: `보증금 + (월세 × 100)`
- 위 결과가 5,000만원 미만이면: `보증금 + (월세 × 70)`
