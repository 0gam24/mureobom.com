# 계산기 명세: 프리랜서 종합소득세

## 1. 개요
- **슬러그**: `/계산기/프리랜서종합소득세`
- **카테고리**: 세금 / 근로
- **타깃 페르소나**: 블루오션 (프리랜서 / 1인 사업자 / N잡러 겸업)
- **1차 키워드**: "프리랜서 종합소득세 계산기"
- **2차 키워드**: 종합소득세 계산, 3.3% 원천징수 환급, 단순경비율 기준경비율
- **우선순위**: MVP #15
- **CPC 티어**: ★★★ ($10-30)

## 2. 사용자 가치
"프리랜서가 1년간 받은 매출·경비·원천징수액으로 5월 종합소득세 신고 시 납부/환급액을 미리 파악"

## 3. 배경

### 3-1. 프리랜서 과세 구조
- 사업소득 (프리랜서 용역 수수료) → 5월 종합소득세 신고
- 지급 시 **3.3% 원천징수** (소득세 3% + 지방소득세 0.3%) — 소득세법 §129, §127
- 연말에 실제 소득세액 계산 후 기납부세액(원천징수액)과 정산
  - 차액 양수 → 추가 납부
  - 차액 음수 → 환급

### 3-2. 필요경비 추정 방식
- **단순경비율**: 소규모 사업자(수입 4,800만 이하, 업종별 상이). 수입금액 × 업종별 단순경비율
- **기준경비율**: 수입 초과 시. 업종별 기준경비율 × 수입 + 실제 주요경비(자료증빙)
- **장부 기장**: 간편장부(수입 7,500만 이하) 또는 복식부기(초과). 실제 경비 공제.

MVP는 **단순경비율 + 실제 경비 직접 입력** 두 방식만 지원. 기준경비율·복식부기는 "세무사 상담" 안내.

### 3-3. 업종별 단순경비율 (2026 기준, 일부 예시)
| 업종 | 단순경비율 |
|---|---|
| 인적용역(94) — 글쓰기·강의·번역·디자인 등 대부분의 프리랜서 | 64.1% |
| IT·컴퓨터 프로그래밍 | 66.6% |
| 학원 강사 | 58% |
| 의료·법무 등 전문직 | 정밀 계산 필요 — MVP 제외 |

(사용자가 직접 경비율을 입력하는 옵션도 제공)

### 3-4. 공제 (MVP 범위)
- 인적공제: 본인 150만 + 부양가족 × 150만
- 국민연금(사업자): 지역가입자 기준, MVP는 직접 입력
- 건강보험(지역가입자): MVP는 직접 입력
- 자녀세액공제: 첫째 15만, 둘째 20만, 셋째+ 각 40만 (연봉실수령액과 동일)

## 4. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 연간 총 수입(매출) | number + 단위버튼 | 원 |
| 경비 산정 방식 | radio | 단순경비율 / 실제 경비 직접 입력 |
| 업종별 단순경비율(%) | number | 기본 64.1 (인적용역) |
| 실제 경비 | number | '실제 경비' 선택 시만 |
| 기납부 원천징수액 (3.3%) | number | 자동 제안(=수입×0.033), 사용자 수정 가능 |
| 부양가족 수 (본인 포함) | number | 기본 1 |
| 20세 이하 자녀 수 | number | 기본 0 |
| 연 국민연금 납부액 | number | 선택, 기본 0 |
| 연 건강보험 납부액 | number | 선택, 기본 0 |

## 5. 결과 출력
- 총 수입
- 필요경비 (경비율 or 직접 입력)
- 사업소득금액 = 수입 − 경비
- 소득공제 합계
- 과세표준
- 산출세액 (소득세법 §55 8단계 누진)
- 자녀세액공제 차감 후 결정세액
- 지방소득세 (결정세액 × 10%)
- 기납부 원천징수액
- **최종 납부(+) 또는 환급(−)액**

## 6. 공식 근거
- 소득세법 §55 (종합소득세율)
- 소득세법 §80 (종합소득과세표준)
- 소득세법 §127·§129 (원천징수 3.3%)
- 소득세법 시행령 §143 (단순경비율)
- 기존 INCOME_TAX_BRACKETS 재사용
