# 계산기 명세: 적금 이자 (정기적금)

## 1. 개요
- **슬러그**: `/계산기/적금`
- **카테고리**: 금융
- **타깃 페르소나**: E (생활 투자자)
- **1차 키워드**: "적금 이자 계산기"
- **2차 키워드**: 단리 복리 차이, 적금 세후 이자, 적금 만기 수령액
- **우선순위**: MVP #11
- **CPC 티어**: ★★★ ($20-45)

## 2. 사용자 가치
"매월 납입 금액·이자율·기간을 입력하면 단리·복리 방식별 만기 수령액과 세후 이자를 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 월 납입금액 | number + 단위버튼 | 원 |
| 연 이자율 (%) | number | 0.01~15, 소수 2자리 |
| 가입 기간(개월) | number | 6·12·24·36·60 중 선택 + 자유입력 |
| 이자 방식 | radio | 단리 / 월복리 |
| 이자 과세 | radio | 일반과세 15.4% / 비과세 / 세금우대 9.5% |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 원금 합계 | 월 납입금액 × 개월수 |
| 세전 이자 | 단리 또는 복리 공식에 의한 세전 이자 |
| 세금 | 이자 × 과세율 (15.4% or 9.5% or 0%) |
| 세후 이자 | 세전 이자 − 세금 |
| **세후 만기 수령액** | 원금 + 세후 이자 |
| 월별 적립 스케줄 요약 | 선택 옵션, 차트 |

## 5. 공식 근거

### 5-1. 단리 정기적금 세전 이자
각 월 납입분이 서로 다른 기간 동안 이자를 받는다.
```
세전 이자 = 월납입금 × 연이자율 / 100 × ( n × (n+1) / 2 ) / 12
n = 개월수
```
(첫 납입분은 n개월 이자, 마지막 납입분은 1개월 이자 → 1..n 합 = n(n+1)/2)

### 5-2. 월복리 정기적금 세전 이자
월이자율 r = 연이자율 / 12 / 100
```
만기 원리금 = 월납입금 × ( (1+r)^n − 1 ) / r × (1+r)
세전 이자 = 만기 원리금 − (월납입금 × n)
```
- 월말 납입(annuity-due 기준) 가정 시 마지막 곱 (1+r) 생략 가능하나 본 계산기는 월초 납입 관행을 따라 (1+r) 가산.
- r === 0 엣지 → 이자 0

### 5-3. 이자소득세
- 일반과세: 이자소득세 14% + 지방소득세(이자세의 10%) 1.4% → 합계 15.4%
- 세금우대: 9.0% + 0.9% → 9.5% (세금우대종합저축, 조세특례제한법 §89의2)
- 비과세: 0% (ISA·청년도약계좌 등 해당 상품)

반올림: 세금·이자 10원 단위 절사.
