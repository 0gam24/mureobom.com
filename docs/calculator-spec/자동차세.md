# 계산기 명세: 자동차세 (승용차)

## 1. 개요
- **슬러그**: `/계산기/자동차세`
- **카테고리**: 생활 / 세금
- **타깃 페르소나**: 블루오션 (자동차 구매자·보유자)
- **1차 키워드**: "자동차세 계산기"
- **2차 키워드**: 자동차세율표, 배기량별 자동차세, 자동차세 연납 할인
- **우선순위**: Phase 2 #4
- **CPC 티어**: ★★★ ($10-25)

## 2. 사용자 가치
"배기량과 차량 연식으로 자동차세·지방교육세·연납 할인까지 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 차량 용도 | radio | 비영업용 승용 / 영업용 승용 / 승합·화물 (MVP는 비영업용 승용만, 나머지는 "전문가 상담") |
| 배기량(cc) | number | 예: 1998 |
| 차량 연식(년) | number | 현재 연도 - 연식 = 경과 연수 |
| 기준 연도 | number | 기본 2026 |
| 연납 할인(1월 납부) 체크 | checkbox | 약 6.4% 할인 (2026 기준) |

## 4. 결과 출력
- 기본 자동차세 (배기량 × cc당 세율)
- 경과 연수별 할인 (3년 초과 연 5%씩, 최대 50%)
- 지방교육세 (자동차세의 30%)
- **연간 납부액**
- 연납 할인 시 총 납부액
- 반기별 납부액 (6월·12월)

## 5. 공식 근거
- 지방세법 §127 (자동차세율)
- 지방세법 §137 (경감)
- 지방세법 §151 (지방교육세, 자동차세의 30%)
- 2026 연납 할인율: 국고예규 (매년 변동, MVP는 6.4% 가정하되 사용자 수정 가능)

## 6. 세율표 (비영업용 승용)

| 배기량 | cc당 세율 |
|---|---|
| 1000cc 이하 | 80원 |
| 1600cc 이하 | 140원 |
| 1600cc 초과 | 200원 |

## 7. 경과 연수별 경감률 (3년차부터 연 5%씩, 최대 50%)
- 1~2년: 0%
- 3년: 5%
- 4년: 10%
- ...
- 12년 이상: 50%
