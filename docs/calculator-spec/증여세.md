# 계산기 명세: 증여세

## 1. 개요
- **슬러그**: `/계산기/증여세`
- **카테고리**: 세금
- **타깃 페르소나**: A (부동산 거래), D (은퇴/상속 준비)
- **1차 키워드**: "증여세 계산기"
- **2차 키워드**: 증여재산공제 10년, 부모자녀 증여 한도, 배우자 증여 6억
- **우선순위**: Phase 2 #1
- **CPC 티어**: ★★★★ ($25-45)

## 2. 사용자 가치
"증여받는 재산가액과 관계(배우자·자녀·기타)만으로 공제 후 납부 세액을 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 증여재산 가액 | number + 단위버튼 | 억 / 천만 / 백만 |
| 증여자 관계 | radio | 배우자 / 성년 직계비속 / 미성년 직계비속 / 직계존속 / 기타 친족 |
| 10년 내 동일 증여자로부터 기증여액 | number | 기본 0 — 합산 과세용 |
| 채무인수 | number | 증여재산에서 차감되는 채무액 (부담부증여) |
| 신고세액공제(3%) 적용 | checkbox | 기한내 신고 시 기본 체크 |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 증여재산가액 | 입력값 |
| 채무인수 | 입력값 |
| 증여세 과세가액 | 가액 - 채무 + 10년내 기증여 합산 |
| 증여재산공제 | 관계별 공제액 |
| 과세표준 | 과세가액 - 공제 |
| 산출세액 | 5단계 누진 |
| 신고세액공제 | 산출세액 × 3% (신고기한 내 한정) |
| **최종 납부세액** | 산출세액 - 신고세액공제 |
| 신고기한 | 증여일 속하는 달 말일 + 3개월 |

## 5. 공식 근거
- 상속세 및 증여세법 §26 (세율)
- 상증세법 §53 (증여재산공제)
- 상증세법 §68 (신고세액공제 3%)
- 상증세법 §47·§55 (과세가액·과세표준)

## 6. 세율표 (상증세법 §26)
| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1억 이하 | 10% | 0 |
| 5억 이하 | 20% | 1,000만 |
| 10억 이하 | 30% | 6,000만 |
| 30억 이하 | 40% | 1억 6,000만 |
| 30억 초과 | 50% | 4억 6,000만 |

## 7. 증여재산공제 (10년 합산, 상증세법 §53)
| 관계 | 공제액 |
|---|---|
| 배우자 | 6억 |
| 성년 직계비속 (자녀·손자) | 5천만 |
| 미성년 직계비속 | 2천만 |
| 직계존속 (부모·조부모) | 5천만 |
| 기타 친족 (형제·며느리 등) | 1천만 |

## 8. 구현 위치
- 공식: `src/lib/tax/gift.ts`

- 추가 상수: `REPORTING_TAX_CREDIT_RATE = 0.03` (신고세액공제 3%, 상증세법 §68)
