## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 대출 금액 | number + 단위버튼 | 억 / 천만 / 백만 |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 월 상환액 | 원리금균등 - 고정값 / 원금균등 - 첫달·마지막달 / 만기일시 - 월 이자만 |
| 총 이자 | 전 기간 이자 합계 |

## 5. 공식 근거

### 5-1. 원리금균등상환 (Amortization)
```
월 상환액 = P × r × (1+r)^n / ((1+r)^n - 1)
P = 대출원금, r = 월이자율(연이자율 ÷ 12 ÷ 100), n = 총 개월수
```
매월 일정 금액 상환. 초기에는 이자 비중 높음, 말기는 원금 비중 높음.

### 5-2. 원금균등상환
```
매월 원금 = P / n (동일)
매월 이자 = 남은 원금 × r (점감)
매월 상환액 = 매월 원금 + 매월 이자
```

### 5-3. 만기일시상환
```
매월 이자 = P × r
원금은 만기에 일시 상환
총 이자 = P × r × n
```

### 5-4. 거치 기간 처리
- 거치 기간 동안에는 이자만 납부(`P × r`)
- 거치 이후 잔여 기간에 대해 위 공식 적용

**출처**: 금융감독원 대출 상환 방식 표준 공식, 상법 §54.
