# 계산기 명세: 재산세 (주택)

## 1. 개요
- **슬러그**: `/계산기/재산세`
- **카테고리**: 세금 / 부동산
- **타깃 페르소나**: A (부동산 거래 직전자), 보유자 전반
- **1차 키워드**: "재산세 계산기"
- **2차 키워드**: 공시가격 재산세, 1세대1주택 재산세 특례, 도시지역분
- **우선순위**: MVP #7
- **CPC 티어**: ★★★★ ($25-50)

## 2. 사용자 가치
"공시가격과 1세대1주택 특례 여부를 입력하면 재산세·지방교육세·도시지역분을 포함한 연 납부액을 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 공시가격 | number + 단위버튼 | 억 / 천만 / 백만 |
| 1세대 1주택 특례 | checkbox | 공시 9억 이하일 때만 적용 |
| 도시지역 | checkbox | 과표 × 0.14% 추가 |
| 지역자원시설세 포함 | checkbox | MVP는 안내만 (계산은 0으로 처리) |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 과세표준 | 공시가격 × 공정시장가액비율 60% |
| 재산세 (본세) | 과표 × 세율 − 누진공제 |
| 특례 적용 여부 | 1세대1주택 공시 9억 이하 시 |
| 도시지역분 | 과표 × 0.14% (도시지역만) |
| 지방교육세 | 재산세 × 20% |
| **총 납부액** | 합계 |
| 납부 일정 | 7월 (1/2) + 9월 (1/2). 연 20만원 이하면 7월 일괄 납부 |

## 5. 공식 근거
- 지방세법 §110 (과세표준)
- 지방세법 §111 (세율) + §111의2 (1세대1주택 특례, 공시 9억 이하)
- 지방세법 §112 (도시지역분) — 과세표준의 0.14%
- 지방세법 §150 (지방교육세) — 재산세의 20%
- 공정시장가액비율 고시 — 주택 60%

## 6. 세율표

### 일반 (다주택·9억 초과 등)
| 과표 | 세율 | 누진공제 |
|---|---|---|
| ~6천만 | 0.1% | 0 |
| ~1.5억 | 0.15% | 3만 |
| ~3억 | 0.25% | 18만 |
| 3억 초과 | 0.4% | 63만 |

### 1세대1주택 특례 (공시 9억 이하)
| 과표 | 세율 | 누진공제 |
|---|---|---|
| ~6천만 | 0.05% | 0 |
| ~1.5억 | 0.1% | 3만 |
| ~3억 | 0.2% | 18만 |
| 3억 초과 | 0.35% | 63만 |
