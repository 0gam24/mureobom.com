# 계산기 명세: D-day (날짜 계산)

## 1. 개요
- **슬러그**: `/계산기/D-day`
- **카테고리**: 생활
- **타깃 페르소나**: 전 사용자
- **1차 키워드**: "D-day 계산기"
- **2차 키워드**: 날짜 차이 계산, 며칠 남았어, 디데이 계산
- **우선순위**: MVP #14

## 2. 사용자 가치
"특정일까지 며칠 남았는지·지났는지, 두 날짜의 차이 일수를 즉시 계산"

## 3. 모드
- **D-day 모드**: 오늘을 기준으로 목표일까지 남은(지난) 일수
- **기간 계산 모드**: 임의의 두 날짜 사이 일수·주·월·년
- **100일/1000일/N일 후 모드**: 기준일 + N일 = 목표일 역산

## 4. 입력 파라미터

### 4-1. D-day 모드
- 기준일 date (기본: 오늘, 사용자 변경 가능)
- 목표일 date

### 4-2. 기간 계산 모드
- 시작일 date
- 종료일 date
- 포함 방식 radio: 양 끝 포함 / 시작일만 포함 / 끝일만 포함 / 제외

### 4-3. N일 후 모드
- 기준일 date (기본: 오늘)
- 일수 number

## 5. 결과 출력
- 남은/지난 일수 (D-XX / D+XX)
- 주 환산 (일수 / 7)
- 개월 환산 (일수 / 30.4167)
- 년 환산 (일수 / 365.25)
- N일 후 모드: 도달 날짜 (YYYY-MM-DD, 요일 포함)

## 6. 공식
- 일수 차이 = UTC 기반 `Math.round((end - start) / 86400000)`
- D-day 표기:
  - 차이 = 0 → "D-DAY"
  - 차이 > 0 → `D-${차이}` (남음)
  - 차이 < 0 → `D+${|차이|}` (지남)
- N일 후: `new Date(UTC(baseDate) + N * 86400000)`
- 요일: ['일','월','화','수','목','금','토']

순수 함수 유지: 현재 날짜는 **UI 계층**에서 주입, `src/lib/utils/dday.ts` 는 날짜 문자열만 받음.
