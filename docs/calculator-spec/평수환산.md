# 계산기 명세: 평수 환산 (평 ↔ 제곱미터)

## 1. 개요
- **슬러그**: `/계산기/평수환산`
- **카테고리**: 부동산 / 생활
- **타깃 페르소나**: A (부동산 거래 직전자), 전 사용자
- **1차 키워드**: "평수 계산기"
- **2차 키워드**: 평수 변환, 제곱미터 평, 전용면적 공급면적, 34평
- **우선순위**: MVP #10
- **CPC 티어**: ★★ ($5-15)

## 2. 사용자 가치
"평 ↔ ㎡ 양방향 변환 + 전용·공급·공용 면적 구분을 한 화면에 제공"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 입력 값 | number | 소수 2자리 |
| 단위 | radio | 평 (py) / 제곱미터 (㎡) |
| 면적 종류 | radio | 전용면적 / 공급면적 / 대지면적 (정보용, 계산에는 영향 없음) |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 변환값 | 입력이 평이면 ㎡로, ㎡이면 평으로 |
| 원주율 표기 | `1평 = 3.3058㎡` / `1㎡ = 0.3025평` |
| 자주 쓰는 평수 변환표 (참조) | 12/18/24/32/34/45/60 평 및 각 ㎡ |

## 5. 공식 근거
- 계량법 시행령 제9조: 국제표준(SI) 단위는 제곱미터(㎡)를 사용한다.
- 1평 = 400/121 m² ≒ 3.3057851239669...㎡ (일본식 전통 척관법 유래, 한국 관습 지속)
- 역변환: 1㎡ = 121/400 평 ≒ 0.30250평
