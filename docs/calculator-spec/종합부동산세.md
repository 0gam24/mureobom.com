# 계산기 명세: 종합부동산세

## 1. 개요
- **슬러그**: `/계산기/종합부동산세`
- **카테고리**: 세금 / 부동산
- **타깃 페르소나**: A (부동산 보유자), D (은퇴층)
- **1차 키워드**: "종합부동산세 계산기"
- **2차 키워드**: 종부세 2026, 1세대1주택 종부세 공제, 공정시장가액비율 60%
- **우선순위**: Phase 2 #3
- **CPC 티어**: ★★★★ ($25-45)

## 2. 사용자 가치
"주택 공시가 합산으로 종부세 과세 대상 여부와 예상 납부세액을 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 보유 주택 수 | radio | 1주택 / 2주택 / 3주택 이상 |
| 보유 주택 공시가 합계 | number + 단위버튼 | 억 / 천만 |
| 1세대1주택자 여부 | checkbox | 1주택 선택 시만 |
| 만 60세 이상 | checkbox | 고령자공제용, 1세대1주택자만 |
| 보유기간(년) | number | 장기보유공제용, 1세대1주택자만 |
| 조정대상지역 주택 포함 | checkbox | 3주택↑ 세율 분기 |

## 4. 결과 출력
- 공시가 합계
- 공제금액 (1세대1주택 12억 / 그 외 9억)
- 종부세 과세표준 = (공시합계 - 공제) × 60% (공정시장가액비율)
- 적용세율 구분 (일반 vs 다주택)
- 종부세 산출세액 (누진)
- 1세대1주택자 세액공제 (고령자·장기보유, 최대 80%)
- **최종 납부세액**
- 농어촌특별세 (종부세의 20%)

## 5. 공식 근거
- 종합부동산세법 §7 (과세표준 = 공시가 − 공제 × 공정시장가액비율)
- 종부세법 §8 (세율)
- 종부세법 §9 (1세대1주택 세액공제: 고령자공제 20~40%, 장기보유 20~50%, 합산 80%)
- 농특세법 (종부세의 20%)

## 6. 세율표 (2026 기준)

### 일반 (1-2주택, 2023 개정)
| 과세표준 | 세율 |
|---|---|
| 3억 이하 | 0.5% |
| 6억 이하 | 0.7% |
| 12억 이하 | 1.0% |
| 25억 이하 | 1.3% |
| 50억 이하 | 1.5% |
| 94억 이하 | 2.0% |
| 94억 초과 | 2.7% |

### 3주택 이상 + 과세표준 12억 초과 (중과)
| 과세표준 | 세율 |
|---|---|
| 3억 이하 | 0.5% |
| 6억 이하 | 0.7% |
| 12억 이하 | 1.0% |
| 25억 이하 | 2.0% |
| 50억 이하 | 3.0% |
| 94억 이하 | 4.0% |
| 94억 초과 | 5.0% |

※ 종부세는 누진구간별 적용(직접세율×누진공제)보다 "각 구간별 초과분 × 세율" 방식 규정. MVP는 `calculateProgressiveTax` 재사용 가능하되 누진공제 값을 각 구간별로 재계산.

## 7. 공제금액
- 1세대1주택자: 12억
- 1세대1주택 특례(부부공동명의 등): 9억 아닌 12억 택일 가능 — MVP는 12억 고정
- 그 외(다주택 포함): 9억
