# 계산기 명세: 취득세

## 1. 개요
- **슬러그**: `/계산기/취득세`
- **카테고리**: 세금 / 부동산
- **타깃 페르소나**: A (부동산 거래 직전자)
- **1차 키워드**: "취득세 계산기"
- **2차 키워드**: 취득세율, 생애최초 감면, 3주택 중과
- **우선순위**: MVP #6

## 2. 사용자 가치
"매매 계약 전 취득세+농특세+지방교육세 합산 총 납부액을 정확히 확정"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 취득방법 | radio | 매매/교환 · 증여 · 상속 · 원시취득 |
| 대상 | radio | 주택 · 농지 · 토지 · 기타 |
| 대상주택수 | radio | 1주택 · 2주택 · 3주택 · 4주택↑ |
| 대상면적 | radio | 85㎡ 이하 · 85㎡ 초과 |
| 조정대상지역 | radio | 예 · 아니오 |
| 취득가액 | number + 단위버튼 | 억/천만/백만/십만 |
| 시가표준액 | number + 단위버튼 | (증여·상속 시) |
| 생애최초 주택 | checkbox | 감면 200만 한도 |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 과세표준 | 취득가 or 시가표준액 |
| 취득세 | 과표 × 세율 |
| 농어촌특별세 | 85㎡ 초과 0.2% (일반) / 1% (중과) |
| 지방교육세 | 취득세 × 10% |
| **총 납부액** | 합계 |
| 납부기한 | 취득일 + 60일 |
