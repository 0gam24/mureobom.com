## 3. 배경: 3대 규제

### 3-1. DSR (Debt Service Ratio, 총부채원리금상환비율)
```
DSR = (모든 대출의 연간 원리금 상환액) / 연소득 × 100
```
- 전 금융권 합산
- 은행: 40%, 2금융권: 50%
- 기준: 주택담보·신용·학자금·자동차 등 모든 대출
- 2024년부터 **스트레스 DSR** 단계 도입:
  - 변동금리·혼합형 대출은 실제 금리에 **스트레스 금리(예: 1.5%p)** 가산해 DSR 산정
  - 2026년 기준 스트레스 금리 1.5%p 적용 가정 (실제 금감원 고시 수치로 교체 가능)

### 3-2. LTV (Loan To Value, 담보인정비율)
```
LTV = 주담대 / 담보가치 × 100
```
- 규제지역(투기·조정) 50%
- 비규제지역 70%
- 서민·실수요자 최대 80%
- 생애최초 최대 80% (2026 기준)
- 9억 초과 주택은 9억 초과분 20-30% 추가 제한

### 3-3. DTI (Debt To Income, 총부채상환비율)
```
DTI = (신규 주담대 연 원리금 + 기존 기타대출 연 이자) / 연소득 × 100
```
- 투기·조정지역 40%
- 기타지역 50-60%
- DSR 전면 시행으로 실효성 낮아졌으나 여전히 병행 적용

## 4. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 연소득 | number + 단위버튼 | 원 |
| 기존 대출 연 원리금 | number | 원 (DSR 계산용) |
| 기존 대출 연 이자 | number | 원 (DTI 계산용, 기본=원리금과 동일) |
| 담보 가치(주택가격) | number + 단위버튼 | 원 |
| 지역 | radio | 비규제 / 조정대상 / 투기·투기과열 |
| 주택 유형 | radio | 일반 / 생애최초·서민실수요 |
| 신규 대출 연이자율 (%) | number | 스트레스 DSR 포함 적용 |
| 신규 대출 기간(년) | number | 10·20·30·40 중 선택 기본 30 |
| 스트레스 DSR 적용 | checkbox | 기본 체크 (변동금리 가정) |
| 상환 방식 | radio | 원리금균등 / 원금균등 / 만기일시 (DSR 산정 방식 영향) |

## 5. 결과 출력

| 항목 | 설명 |
|---|---|
| DSR 기준 한도 | DSR 40% 또는 50% 한도 내 월 상환 가능액 → 총 대출가능원금 |
| LTV 기준 한도 | 담보가치 × LTV 비율 |
| DTI 기준 한도 | DTI 비율로 역산 |
| **최종 한도** | 3 기준 중 **최솟값** |
| 월 예상 상환액 | 최종 한도 기준 |
| 적용 스트레스 금리(%) | 체크 시 |
| 초과분 경고 | 담보가치에 비해 DSR·DTI 더 타이트하게 걸리면 안내 |
