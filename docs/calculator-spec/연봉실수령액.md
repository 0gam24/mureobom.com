# 계산기 명세: 연봉실수령액

## 1. 개요
- **슬러그**: `/계산기/연봉실수령액`
- **카테고리**: 근로
- **타깃 페르소나**: C (직장인)
- **1차 키워드**: "연봉 실수령액 계산기"
- **2차 키워드**: 세후 월급, 4대보험, 소득세 월급
- **우선순위**: MVP #1 (고검색량 + 고CPC)

## 2. 사용자 가치
"연봉 협상·이직 제안 시 세후 실수령액이 얼마인지 1분 안에 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 기본값 | 유효 범위 | 설명 |
|---|---|---|---|---|
| 임금유형 | radio | 연봉 | 연봉 / 월급 | |
| 연봉액 | number | - | 0 - 100억 | 세전 연봉 (원) |
| 월급액 | number | - | 0 - 10억 | 임금유형=월급일 때 |
| 퇴직금 | radio | 별도 | 별도 / 포함 | 연봉에 퇴직금 포함 여부 |
| 비과세액(월) | number | 0 | 0 - 200만 | 식대 등 비과세 수당 월액 |
| 부양가족수 | number | 1 | 1 - 20 | 본인 포함 |
| 자녀수 | number | 0 | 0 - 10 | 20세 이하 자녀 |

## 4. 결과 출력

| 항목 | 설명 | 시각화 |
|---|---|---|
| 예상 월 소득(세전) | = 연봉/12 | 숫자 |
| 국민연금 | 월소득 × 4.5% (상한 40~637만 기준) | 숫자 |
| 건강보험 | 월소득 × 3.545% | 숫자 |
| 장기요양 | 건보료 × 12.95% | 숫자 |
| 고용보험 | 월소득 × 0.9% | 숫자 |
| 근로소득세 | 간이세액표 | 숫자 |
| 지방소득세 | 소득세 × 10% | 숫자 |
| **월 실수령액** | 월소득 - 공제 합 | **hero 숫자** |
| 시급 (월 209h 기준) | 실수령액 / 209 | 부가 |
| 중위소득 대비 % | 기준중위소득 × 300%까지 누진표시 | 도넛 |

## 5. 공식 근거
- 소득세법 §55 (세율)
- 소득세법 §59의2 (근로소득공제)
- 국민건강보험법 (건보료율)
- 국민연금법 (연금요율)
- 국세청 근로소득 간이세액표 2026
