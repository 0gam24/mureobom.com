## 3. 배경: 전환율 규제 (주택임대차보호법 §7의2)

전세보증금을 월세로 전환 시 한도는 아래 중 **낮은 비율**:
- (A) 한국은행 기준금리 + 대통령령 비율(현재 2%p)
- (B) 연 10%

2026년 기준 예시: 한국은행 기준금리 3.5% → A = 5.5%, B = 10% → 실제 한도 5.5%.
※ 한국은행 기준금리는 시점에 따라 변동. 상수로 두되 사용자가 수정 가능하게.

## 4. 모드

### 4-1. 전세 → 월세 변환
- 기존 전세보증금
- 감소한 보증금 (또는 새 보증금)
- 결과: 월세 = (전세 - 새보증금) × 전환율 / 12

### 4-2. 월세 → 전세 변환
- 기존 보증금
- 월세
- 결과: 환산 전세 = 보증금 + 월세 × 12 / 전환율

### 4-3. 전환율 자체 계산
- 전세보증금
- 반전세 보증금 + 월세
- 결과: 실제 적용된 전환율(%) = 월세 × 12 / (전세 - 새보증금) × 100

## 5. 입력 파라미터 (공통)
- 한국은행 기준금리(%) — 기본 3.5
- 대통령령 가산비율(%p) — 기본 2.0
- 연 상한(%) — 기본 10.0

## 6. 결과 출력
- 적용 전환율(%)
- 모드별 변환 금액
- 법정 상한 초과 여부 경고
- 환산보증금(월세×100 포함) 참고 표시

## 7. 공식 근거

- 한국은행 기준금리 (매 분기 확인, 2026년 기준 3.5% 기본값)
