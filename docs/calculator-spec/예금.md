# 계산기 명세: 예금 이자 (정기예금)

## 1. 개요
- **슬러그**: `/계산기/예금`
- **카테고리**: 금융
- **타깃 페르소나**: E (생활 투자자)
- **1차 키워드**: "정기예금 이자 계산기"
- **2차 키워드**: 예금 세후 이자, 단리 복리 예금
- **우선순위**: MVP #12
- **CPC 티어**: ★★★ ($15-30)

## 2. 사용자 가치
"목돈을 예치할 때 단리·복리 방식, 세전·세후 이자, 만기 수령액을 즉시 비교"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 예치 원금 | number + 단위버튼 | 억 / 천만 / 백만 |
| 연 이자율 (%) | number | 0.01~15 |
| 예치 기간(개월) | number + 퀵버튼 | 3·6·12·24·36·60 |
| 이자 방식 | radio | 단리 / 월복리 / 일복리 |
| 이자 과세 | radio | 일반과세 15.4% / 비과세 / 세금우대 9.5% |

| 항목 |

| 세전 이자 |

| **세후 만기 수령액** |
| 연환산 세후 이자율 |

## 5. 공식

### 5-1. 단리
```
이자 = 원금 × (연이자율/100) × 개월/12
```

### 5-2. 월복리
```
월이자율 r = 연이자율/100/12
만기원리금 = 원금 × (1+r)^n
이자 = 만기원리금 − 원금
```

### 5-3. 일복리 (근사)
```
일이자율 r = 연이자율/100/365
총일수 d = 개월 × 30.4167 (근사) 또는 실제 일수 입력 — MVP 는 30.4167 고정
만기원리금 = 원금 × (1+r)^d
이자 = 만기원리금 − 원금
```

### 5-4. 세금
적금과 동일 (src/lib/constants/savings-rates-2026.ts 재사용).
