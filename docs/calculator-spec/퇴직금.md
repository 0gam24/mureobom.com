# 계산기 명세: 퇴직금 (법정 퇴직금 + 퇴직소득세)

## 1. 개요
- **슬러그**: `/계산기/퇴직금`
- **카테고리**: 근로
- **타깃 페르소나**: C (직장인), D (은퇴 준비층)
- **1차 키워드**: "퇴직금 계산기"
- **2차 키워드**: 퇴직소득세, DB DC 비교, 평균임금 퇴직금
- **우선순위**: MVP #2
- **CPC 티어**: ★★★ ($15-25)

"퇴직·이직·정년 전에 1일 평균임금 기반 법정 퇴직금과 세후 수령액을 정확히 파악"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 입사일 | date | - |
| 퇴사일 | date | - |
| 퇴직 전 3개월 월 통상임금 | number | 원 (혹은 월평균값) |
| 퇴직 전 3개월 기타 수당 총액 | number | 기본 0 (주휴·야간·연장수당 등) |
| 연간 상여금 총액 | number | 연 1회분 전체, 기본 0 |
| 연차수당 (연간) | number | 기본 0 |
| 퇴직연금 제도 | radio | 법정퇴직금(일반) / DB형 / DC형 |
| 세금 계산 | radio | 포함 / 제외 |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 재직일수 | 퇴사일 - 입사일 (+1) |
| 재직 연수 | 재직일수 / 365, 소수점 2자리 |
| 1일 평균임금 | 3개월 임금총액 / 91-92일 (구체 일수 계산) |
| 법정 퇴직금 | 평균임금 × 30 × 재직일수/365 |
| 퇴직소득세 | 환산급여 방식 산출 |
| 지방소득세 | 퇴직소득세 × 10% |
| **세후 실수령액** | 법정 퇴직금 - 세금 |
| DB/DC 비교 참고 | DB형은 법정 퇴직금 동일. DC형은 사업주 부담금 누적 + 운용수익 (본 계산기는 법정 기준) |

## 5. 공식 근거

### 5-1. 평균임금 (근로기준법 §2)
```
1일 평균임금 = (산정사유 발생일 이전 3개월 간의 임금총액) / (그 기간의 총일수)
임금총액 = 3개월 통상임금 + (연 상여금 × 3/12) + (연차수당 × 3/12) + 기타 수당
```

### 5-2. 법정 퇴직금 (근로자퇴직급여 보장법 §8)
```
퇴직금 = 1일 평균임금 × 30일 × (재직일수 ÷ 365)
```
- 1년 이상 근무한 근로자에게 지급
- 재직일수 1년 미만은 지급 의무 없음

### 5-3. 퇴직소득세 (소득세법 §55·§148)
근속연수공제 (2023 세법개정 이후, 2026 유지):
| 근속연수 | 공제액 |
|---|---|
| 5년 이하 | 근속연수 × 100만원 |
| 5년 초과~10년 | 500만 + (근속연수 − 5) × 200만 |
| 10년 초과~20년 | 1,500만 + (근속연수 − 10) × 250만 |
| 20년 초과 | 4,000만 + (근속연수 − 20) × 300만 |

환산급여 공식:
```
환산급여 = (퇴직소득금액 − 근속연수공제) × 12 ÷ 근속연수
```

환산급여공제 (단계별):
| 환산급여 구간 | 공제액 |
|---|---|
| 800만 이하 | 전액 |
| 800만~7,000만 | 800만 + (환산급여 − 800만) × 60% |
| 7,000만~1억 | 4,520만 + (환산급여 − 7,000만) × 55% |
| 1억~3억 | 6,170만 + (환산급여 − 1억) × 45% |
| 3억 초과 | 1억 5,170만 + (환산급여 − 3억) × 35% |

과세표준 = 환산급여 − 환산급여공제
산출세액 = (과세표준 × 기본세율) ÷ 12 × 근속연수

기본세율은 소득세법 §55 종합소득세율 테이블 재사용.
