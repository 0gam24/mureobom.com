# 계산기 명세: 양도소득세

## 1. 개요
- **슬러그**: `/계산기/양도소득세`
- **카테고리**: 세금 / 부동산
- **타깃 페르소나**: A (부동산 거래 직전자)
- **1차 키워드**: "양도소득세 계산기"
- **2차 키워드**: 일시적 2주택, 1세대1주택 비과세, 장기보유특별공제
- **우선순위**: MVP #5

## 2. 사용자 가치

## 5. 공식 근거
- 소득세법 §92-§118
- 시행령 §154 (일시적 2주택 비과세)
- 시행령 §159의3 (장기보유특별공제)
- 소득세법 §104의3 (조정지역 중과)

## 6. 복잡성 주의사항
양도소득세는 본 프로젝트 가장 복잡한 계산기. 다음 특례 분기 필수:
- 1세대1주택 비과세 (12억 이하 전액)
- 1세대1주택 고가주택 (12억 초과분만 과세)
- 일시적 2주택 3년 이내
- 상속주택 특례
- 농어촌주택 특례
- 장기임대등록주택 특례

초기 버전은 **일반 + 1세대1주택 + 일시적2주택** 3케이스만. 나머지는 "전문가 상담 권장" 안내.
