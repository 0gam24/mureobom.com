# 계산기 명세: BMI (체질량지수)

## 1. 개요
- **슬러그**: `/계산기/BMI`
- **카테고리**: 생활 / 건강
- **1차 키워드**: "BMI 계산기"
- **2차 키워드**: 체질량지수, 비만도 계산, 표준체중, 정상 BMI
- **우선순위**: MVP #13

## 2. 사용자 가치
"키와 몸무게로 BMI 값과 대한비만학회 기준 체중 분류를 즉시 확인"

## 3. 입력 파라미터

| 필드 | 타입 | 범위 |
|---|---|---|
| 키 | number | cm (100~250) |
| 몸무게 | number | kg (20~300) |
| 성별 | radio | (선택사항) 남/여 (표준체중 참고용) |
| 나이 | number | (선택사항) 참고 메모용 |

## 4. 결과 출력
- **BMI 수치** (소수 2자리)
- **분류** — 대한비만학회 기준:
  - ~18.4: 저체중
  - 18.5~22.9: 정상
  - 23.0~24.9: 과체중 (비만 전단계)
  - 25.0~29.9: 1단계 비만
  - 30.0~34.9: 2단계 비만
  - 35.0~: 3단계 비만 (고도비만)
- **표준 체중 권장 범위**: BMI 18.5~22.9에 해당하는 kg 범위
- 목표 체중 이상/이하 (현재 대비 증감 필요량)

## 5. 공식 근거
- BMI = 몸무게(kg) / 키(m)²
- 분류 기준: 대한비만학회 「비만 진료지침」 2022 개정판 (한국 기준)
- WHO 기준(서양)과 차이: WHO는 과체중 25+, 비만 30+ (아시아는 더 엄격)
