# 계산기 명세: 상속세

## 1. 개요
- **슬러그**: `/계산기/상속세`
- **카테고리**: 세금
- **타깃 페르소나**: D (은퇴 준비층), A (부동산 보유자)
- **1차 키워드**: "상속세 계산기"
- **2차 키워드**: 일괄공제 5억, 배우자 상속공제, 상속세 5단계 세율
- **우선순위**: Phase 2 #2
- **CPC 티어**: ★★★★ ($25-45)

## 2. 사용자 가치
"상속재산 규모·상속인 구성으로 공제 후 납부세액을 즉시 확인, 자녀·배우자 유무 시나리오별 비교"

## 3. 입력 파라미터

| 필드 | 타입 | 옵션 |
|---|---|---|
| 상속재산 총액 | number + 단위버튼 | 억 / 천만 / 백만 |
| 장례비 등 공과금 | number | 기본 0 (상증세법 §14) |
| 배우자 상속 여부 | checkbox | 기본 체크 |
| 배우자 실제 상속분 | number | 배우자 있을 때만 입력, 기본 0 |
| 자녀 수 | number | 20세 이하 자녀 별도 |
| 미성년 자녀 수 | number | |
| 일괄공제 사용 (5억) | radio | 기초공제+기타인적공제 vs 일괄공제 5억 중 유리한 것 자동 적용 / 수동 선택 |
| 기한내 신고 | checkbox | 신고세액공제 3% |

## 4. 결과 출력

| 항목 | 설명 |
|---|---|
| 상속재산 총액 |
| 공과금·장례비 |
| 총 상속재산가액 | 재산 - 공과금 |
| **상속공제 합계** | 각 공제 합산 |
|  ├ 기초공제 | 2억 |
|  ├ 자녀공제 | 5천만 × 자녀수 |
|  ├ 미성년자공제 | 1천만 × (19세 - 나이) × 미성년자녀수 (MVP 근사) |
|  ├ 일괄공제 | 5억 (위 합보다 유리하면) |
|  └ 배우자공제 | 법정상속분 내 실제 상속액 (최소 5억, 최대 30억) |
| 과세표준 | 총상속재산 - 공제 |
| 산출세액 | 5단계 누진 (증여세와 동일) |
| 신고세액공제 | 3% |
| **최종 납부세액** | |
| 신고기한 | 상속개시일 + 6개월 |

## 5. 공식 근거
- 상속세 및 증여세법 §18 (기초공제)
- 상증세법 §20 (기타인적공제: 자녀·미성년·장애·65세 이상)
- 상증세법 §21 (일괄공제 5억)
- 상증세법 §19 (배우자 상속공제)
- 상증세법 §26 (세율) — 증여세와 공통
- 상증세법 §68 (신고세액공제 3%)

## 6. 공제 상수 (신규 tax-rates-2026.ts 추가)
- `INHERITANCE_BASIC_DEDUCTION = 200_000_000` (기초공제 2억)
- `INHERITANCE_LUMP_SUM_DEDUCTION = 500_000_000` (일괄공제 5억)
- `INHERITANCE_CHILD_DEDUCTION_PER_HEAD = 50_000_000` (자녀공제 5천만/인)
- `INHERITANCE_MINOR_DEDUCTION_PER_YEAR = 10_000_000` (미성년자공제 1천만/연)
- `INHERITANCE_SPOUSE_MIN_DEDUCTION = 500_000_000` (배우자 최소 5억)
- `INHERITANCE_SPOUSE_MAX_DEDUCTION = 3_000_000_000` (배우자 최대 30억)

(세율 5단계 `GIFT_INHERITANCE_TAX_BRACKETS` 재사용, `REPORTING_TAX_CREDIT_RATE` 재사용)
