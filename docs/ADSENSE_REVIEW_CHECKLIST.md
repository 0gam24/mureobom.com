# 애드센스 심사 위생 체크리스트 (Phase 0)

`/gate-check --hygiene` 실행 시 이 문서를 기준으로 사이트 전체를 점검한다.

## 필수 페이지
- [ ] 개인정보처리방침 (쿠키·광고 식별자 언급 포함)
- [ ] 운영자 소개 / About (사이트 목적·운영 주체)
- [ ] 문의 페이지 또는 이메일 (footer에서 접근 가능)
- [ ] (있다면) 제휴·광고 고지 문구

## 콘텐츠 품질
- [ ] 300자 미만의 얇은 글 0개 (있으면 보강 또는 noindex)
- [ ] 복사·짜깁기 흔적 없는지 표본 점검 (최근 글 5개)
- [ ] 모든 글에 출처와 기준일 표기
- [ ] 깨진 이미지·깨진 내부링크 0개

## 구조·기술
- [ ] 카테고리 네비게이션이 명확하고 빈 카테고리 없음
- [ ] sitemap.xml 제출 + robots.txt 정상
- [ ] 모바일 렌더링 정상 (Q&A 포맷의 표가 모바일에서 깨지지 않는지)
- [ ] 페이지 속도: 치명적 차단 리소스 없음
- [ ] 404 페이지 존재

## 심사 중 금지
- [ ] 사이트 구조 대규모 변경 금지
- [ ] 발행량 급증 금지 (일 3개 유지)
- [ ] 타 광고 네트워크 코드 동시 부착 금지
