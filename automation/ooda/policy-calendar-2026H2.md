## 운영 메모

- **재공유 = GSC URL 검사 + Naver SA 웹페이지 수집** 동시 실행. SNS·카페 공유는 부수효과.
- **같은 글 N회 재공유**: 같은 해 안에 2~3회 재요청 가능 (검색량 사이클이 명확한 글).
- **트래픽 점프 1차 레버**: 트랙 2(재공유)가 트랙 1(신규)보다 ROI 높음 — 신규는 색인까지 며칠, 재공유는 즉시 노출 복귀.
- **GSC·Naver SA 측정 우선**: 어떤 글이 색인은 됐는데 노출 0인지 먼저 확인 → 재공유 우선순위 재조정.
- **신규 발행은 [calendar-2026-06.md](calendar-2026-06.md)와 본 캘린더 시즌 슬롯 우선** — 6/25 이후 7~8월 슬롯이 비어있으니 채워야 함.
- 다음 OODA 결정 후보:
  - `D-2026-06-12-N`: 트랙 2(재공유) 자동화 메커니즘 — GSC URL 검사 API 활용
  - `D-2026-06-12-N+1`: 신규 발행 캐던스를 시즌 기반으로 재조정 (현 loan/tax 위주 → 시즌 동적)
